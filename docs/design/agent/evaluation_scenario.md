# Agent 记忆与推理能力测评场景

## 场景设计说明

本场景模拟一位互联网从业者（陈默）从 2018 年到 2026 年的职业与生活轨迹，包含：
- **时序推理**：多段工作经历、项目时间线、生活事件的时间顺序
- **多跳推理**：人物关系网络、公司间关联、间接因果关系
- **因果推理**：离职原因、创业动机、决策背后的驱动因素
- **复杂任务关系**：项目中的多方协作、任务依赖、并行工作流

## 推理问题集

### 一、时序推理（Temporal Reasoning）

| 编号 | 问题 | 答案 | 推理要点 |
|------|------|------|----------|
| T1 | 陈默在阿里巴巴和字节跳动分别工作了多长时间？哪家时间更长？ | 阿里：2年2个月（2018.7-2020.9）；字节：10个月（2020.10-2021.8）。阿里更长。 | 需计算两段工作时间长度并比较 |
| T2 | 智云科技从成立到产品正式上线间隔了多久？ | 9个月（2021.9成立，2022.6上线） | 需定位两个时间点并计算差值 |
| T3 | 陈默儿子出生时，他正处在职业生涯的哪个阶段？ | 儿子2023年3月出生，当时陈默在智云科技任CTO，公司正面临A轮融资困难。 | 需将生活事件与职业事件对齐到同一时间轴 |
| T4 | 按时间顺序排列以下事件：①甲状腺结节体检 ②字节技术卓越奖 ③智云Pre-A轮融资 ④结婚 | ①（2020.8）→ ②（2021.3）→ ④（2022年）→ ③（2022.1，实际结婚在2022年，但Pre-A是2022.1，需精确判断：Pre-A 2022.1，结婚在2022年但月份未知，根据上下文推断结婚应在Pre-A之后，因为公司成立后他才结婚） | 多事件时间排序，部分需要间接推断 |
| T5 | 陈默父亲做手术时，智云科技的B轮融资完成了吗？ | 父亲2025.5手术，B轮2024.9已完成。答案是：已完成。 | 判断两个事件的前后关系 |
| T6 | 从陈默离开字节到智云产品上线，中间经历了哪些公司/阶段？按顺序列出。 | 字节（2020.10-2021.8）→ 智云筹备期（2021.8-2021.9）→ 智云成立后产品开发期（2021.9-2022.6） | 追踪跨公司的时间连续性 |
